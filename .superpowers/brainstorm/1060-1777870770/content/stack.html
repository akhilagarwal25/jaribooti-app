Backend architecture

Jaribooti uses Shopify. But we still need a small backend for login/auth and cart.

A

Shopify-only (Recommended)

No separate backend. App talks directly to Shopify Storefront API for products, cart, checkout. A thin wrapper if needed. Simplest architecture.

B

Django backend (like Andha Pansari)

Separate Django app on your VPS. Shopify handles products/checkout. Django handles auth, cart sync, notifications, order webhooks, analytics.
