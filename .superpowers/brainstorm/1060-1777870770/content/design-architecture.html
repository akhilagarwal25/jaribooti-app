Tech stack

  • Expo SDK 54 + expo-router v6 + NativeWind v4 (same as Andha Pansari app)
  • Shopify Storefront API — products, collections, cart, checkout URL
  • Shopify Checkout — WebView with postMessage bridge for order confirmation
  • Auth — Shopify customer account + Google OAuth + Phone OTP
  • Green botanical theme — matching jaribooti.com branding

Key screens

  • Home — hero banner, featured products, collections
  • Catalog / Collections — browse by category
  • Product detail — images, description, variants, add to cart
  • Search — product search with filters
  • Cart — line items, quantity, proceed to checkout
  • Checkout — WebView to Shopify hosted checkout
  • Order confirmation — postMessage from WebView, show success
  • Account — login/register, order history, saved addresses
  • Profile — edit details, logout
